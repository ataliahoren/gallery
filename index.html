Photo-gallery

picture sharing project as part of cloud computing course - include api backend and client web.

Description

  • PHOTO Sharing Gallery
  • backend api
  • web client

What is the project about and include

  • Highly available, RESTful API that is accessible from internet, and hosted on AWS.
  • Trafﬁc is load-balanced using ELB
  • Ofﬂine processing will is done by background worker
  • Image metadata is stored on MongoDB (use mLAB)
  • Image binary data is stored on S3
  • Cache layer is provided by Redis
  • Web client demo (1 html ﬁle, and js) that demonstrate the API.

Project Functionallity

  • Support for posting of new binary images
  • Support for replacing existing images
  • Support for deleting of existing image
  • Support for adding metadata to existing image (creator, title)
  • Ofﬂine processing of posted image - resizing (original image, thumbnail, web size) , extraction of dominant color
  • Support for querying for all images and according to dominant color
  • Web client - Gallery and CRUD provider (Create, Update, Delete) for images
